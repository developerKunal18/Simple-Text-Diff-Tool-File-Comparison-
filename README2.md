# Simple Text Diff Tool

A Python script that compares two text files and shows differences.

## Description
This project reads two files and displays which lines were added or removed.

## Features
- File comparison
- Detects added lines
- Detects removed lines
- Useful for debugging and version tracking

## Technologies Used
- Python
- difflib module

## How to Run
1. Clone the repository
2. Place two text files in the project folder
3. Run:

   python file_diff.py

4. Enter file names

## Example

Output:
Removed: Hello world  
Added: Hello Python
